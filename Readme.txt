Run For Your Life est un platformer axé speed run développé par OCTOGONE GAMES©.

Histoire :

Vous venez de vous lever et votre réveil viens de sonner !
Vite pas de temps à perdre ! Pas le temps de s'habiller vous partez au travail et vous n'avez que 3 minutes.
Parcourez un monde étrange et semé d'embûche pour atteindre votre lieu de travail.


Deux modes de jeu :

Story : Faîtes les différentes map dans l'ordre avec un timer de 3 minutes et tenter d'obtenir le meilleur score
Freeplay : Les maps du mode story en mode aléatoire, pas de timer, idéal pour s'entrainer avant d'essayer de faire votre meilleur temps.

Touches :

Saut = ESPACE
Déplacement = FLECHE DIRECTIONELLES
Dash = A ou Q (EU ou US)
Restart le niveau = R
Story/Freeplay : Restart la run/Map aléatoire = SUPPR
